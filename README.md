# Peculiar Site 2.0

A web project built with JavaScript, HTML, and CSS.

## Overview

This is version 2.0 of the Peculiar Site project, featuring a modern web experience with interactive functionality and responsive design.

## Technology Stack

- **JavaScript** (45.1%) - Core interactive functionality
- **HTML** (39.2%) - Semantic markup and structure
- **CSS** (15.7%) - Styling and responsive design

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Basic understanding of HTML, CSS, and JavaScript

### Installation

1. Clone the repository:
```bash
git clone https://github.com/prashant-pandit/peculiar-site-2.0.git
cd peculiar-site-2.0
```

2. Open the project in your preferred code editor or simply open `index.html` in your browser.
